<h1>Desenvolvendo aplicações Windows Forms com C#!</h1>
<p>Dentre as tecnologias disponíveis atualmente, o ecossistema .NET é valorizado e reconhecido no mercado pela estabilidade, confiabilidade, produtividade, pelas APIs maduras e funcionalidades modernas, ferramentas eficientes, documentação vasta e o suporte de primeira linha oferecido pela Microsoft.

<h2>O que é Windows Forms?</h2>
O Windows Forms é o sucessor do Visual Basic Forms e possibilita a criação da aplicações Desktop para Windows é uma linguagem criada para o Microsoft .NET Framework e .NET Core, e foi projetado para a criação de aplicativos seguros e orientados a objetos, de forma produtiva e eficaz.

<h2>O que aprendi e o que estou aprendendo ao longo desta Formação?</h2>
A Formação Windows Forms com C# da Alura proporciona uma abordagem prática de aprendizado para a/o profissional que quer se destacar e estar preparada(o) para os desafios do mercado, através da bagagem teórica e experiências práticas adquiridas ao longo dos cursos da formação.

Você vai aprender a criar aplicações Desktop com Windows Forms usando a linguagem C# da MicroSoft construindo uma interface rica e completa. Usaremos o SQL Server para a persistência e aprendemos todos os padrões para criar uma aplicação robusta.</p>
